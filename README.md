A arquitetura utiliza o **Playwright** como biblioteca de automacao, o **Mocha** como executor de testes, e o **GitHub Actions** para orquestrar a execucao e exibir relatorios visuais diretamente na plataforma.echo.echo ---echo.echo ## ??? Tecnologias e Conceitos Utilizadosecho.echo *   **Playwright (Library Mode):** Utilizado para controlar navegadores de forma rapida e confiavel. Diferente do modo tradicional (@playwright/test), aqui ele e gerenciado programaticamente dentro de scripts Node.js puro.echo *   **Mocha:** Framework de testes responsavel por estruturar os cenarios (describe, it) e gerenciar o ciclo de vida da execucao.echo *   **xUnit / JUnit XML (Formato de Relatorio):** Um padrao universal de arquivos XML utilizado para descrever os resultados de testes automatizados. E o formato que permite a integracao com ferramentas de terceiros e com o proprio GitHub.echo *   **GitHub Actions (CI/CD):** Ferramenta de automacao nativa do GitHub utilizada para rodar os testes a cada alteracao de codigo ou de forma agendada.echo *   **Artefatos de Build (Artifacts):** Arquivos gerados durante a execucao da pipeline que sao salvos pelo GitHub para download posterior (neste caso, o relatorio XML).echo.echo ---echo.echo ## ?? Como Funciona a Solucao (Fluxo da Pipeline)echo.echo O arquivo de configuracao do GitHub Actions (.github/workflows/e2e-tests.yml) segue uma sequencia logica projetada para garantir eficiencia e visibilidade:echo.echo ```echo [Gatilho] => [Setup do Ambiente] => [Instalacao] => [Execucao] => [Geracao de Relatorios]echo ```echo.echo 1. **Gatilhos (Triggers):** A pipeline e ativada automaticamente por execucao manual, push na branch `main` ou de forma agendada todas as sextas-feiras as 09:00 AM.echo 2. **Ambiente Isolado (Runner):** O trabalho roda em um container limpo com **Ubuntu 24.04** e **Node.js v22.x LTS**.echo 3. **Instalacao Limpa (npm ci):** Garante a instalacao exata das dependencias do projeto com base no arquivo `package-lock.json`.echo 4. **Instalacao dos Navegadores:** O comando `npx playwright install --with-deps` baixa os binarios do Chromium, Firefox e Webkit.echo 5. **Geracao Dinamica do Reporter:** A pipeline instala o `mocha-junit-reporter` localmente antes da execucao.echo 6. **Execucao e Captura (if: always()):** Mesmo que um teste quebre e o comando `npm test` falhe, os passos seguintes de relatorio sao forcados a rodar.echo.echo ---echo.echo ## ??
